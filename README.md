# Optimize Configurables
Magento Extension to optimize store with huge number of Configurable Products and Configurable Options.

Install (modman)
------------------
    $ cd path/to/magento
    $ modman init
    $ modman clone https://github.com/deniscsz/optimize-configurables.git
